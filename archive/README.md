## build-loop-auto-research (retired 2026-04-17)

Karpathy's autoresearch pattern adapted for app development. Functionality folded into `build-loop` v0.2.0+:
- Optimization loop → `build-loop:optimize` (Phase 4.7)
- Research packets → `build-loop:research` (Phase 1 / standalone)
- Performance tracking → `scripts/experiment_metrics.py`

Original repo had: research packets, history analysis, brief optimization, context snapshots, lesson memory. The optimization loop and research packets were the valuable parts; the rest overlapped with bookmark and build-loop's memory system.
