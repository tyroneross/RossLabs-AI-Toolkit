# Build Loop Research Memory

Use this index to track modular lesson notes for situations that needed multiple iterations, failed because of hidden complexity, or broke at integration / handoff boundaries.

## When To Capture A Lesson

- the first recommendation was too complex
- confidence stayed `medium` or `low` after an iteration
- an API, auth, payment, or deployment assumption was wrong
- a handoff between modules, services, or environments failed
- the simplest approach was not chosen early enough
