# Agents

Agent configurations for autonomous and semi-autonomous tasks.

## Available Agents

*No standalone agents yet. Agents bundled within plugins are documented in their respective plugin repos.*

### Plugin-Bundled Agents

| Agent | Plugin | Purpose |
|-------|--------|---------|
| `snapshot-analyst` | Bookmark | Analyze session snapshots |
| `api-assessor` | Claude Code Debugger | API endpoint diagnosis |
| `database-assessor` | Claude Code Debugger | Database/query diagnosis |
| `frontend-assessor` | Claude Code Debugger | React/frontend diagnosis |
| `performance-assessor` | Claude Code Debugger | Latency/performance diagnosis |
| `assessment-orchestrator` | Claude Code Debugger | Coordinate parallel assessors |
| `visual-iterator` | IBR | Iterative UI refinement loop |
| `architecture-advisor` | NavGator | Architecture recommendations |
| `architecture-investigator` | NavGator | Deep architecture analysis |
